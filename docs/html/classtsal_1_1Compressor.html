Detailed Description

An audio compressor.

A compressor used to reduce clipping. Essentially, if the sound routed through the compressor is louder than the compressors threshold, it will reduce the loudness. Implementation adapted from https://christianfloisand.wordpress.com/2014/06/09/dynamics-processing-compressorlimiter-part-1/
