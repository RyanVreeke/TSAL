Detailed Description

A SoundFont 2 synthesizer that can sf2 files.

SoundFonts seem to be the easiest way to synthesize real instruments. The SoundFont class uses the TinySoundFont library to load and play sf2 files.

Useful links:

  • https://github.com/schellingb/TinySoundFont/
  • http://www.synthfont.com/soundfonts.html
