Detailed Description

An sound envelope that can be used to change a sound over time.

A 4 stage envelope transitions from attack, to decay, to sustain, and then to release. It achieves a smoother sound when overlayed on an instrument Implementation adapted from http://www.martin-finke.de/blog/articles/audio-plugins-011-envelopes/
